# kekedubing

Local AI video translation, dubbing, subtitle burning, and simple blur editing.

`kekedubing` is an open-source local video dubbing app powered by Python, FastAPI, faster-whisper, Argos Translate, Supertonic, FFmpeg, and yt-dlp. It helps creators translate videos, generate dubbed audio, burn styled subtitles, import supported video URLs, and export ready-to-share MP4 files without depending on cloud APIs by default.
